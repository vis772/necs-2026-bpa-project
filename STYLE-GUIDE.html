Design System v1.0

NECS 2026 Style Guide

This comprehensive design system defines the visual language for the National Esports Championship Series 2026. It ensures consistency across all touchpoints including the promotional website, mobile application, and marketing materials.

Brand Overview

Design Philosophy

The NECS 2026 brand embodies the energy, precision, and excitement of competitive esports. Our design language combines dark, immersive backgrounds with vibrant neon accents to create an atmosphere that feels both premium and electric.

Key Principles: Modern, Bold, Accessible, Consistent, Performance-Focused

Accessibility

Color Contrast

All text colors have been tested to meet WCAG 2.1 AA standards for contrast ratios:

  • ✓ Primary text (#ffffff) on dark backgrounds: 15.5:1 ratio
  • ✓ Secondary text (#a1a1aa) on dark backgrounds: 7.2:1 ratio
  • ✓ Accent cyan (#00f0ff) on dark backgrounds: 11.8:1 ratio

Interactive Elements

  • ✓ All interactive elements have visible focus states
  • ✓ Touch targets meet minimum 44x44px requirement
  • ✓ Color is not the only indicator of state changes
  • ✓ Animations respect prefers-reduced-motion
